# Pitch your startup to AI investors

## Claude can generate complete business simulation applications that combine startup evaluation, investor psychology, scoring systems, interactive workflows, and modern user interfaces. This exercise demonstrates how AI can be used to create startup education tools and interactive business experiences.
    
    1 Investor Thinking: Understand how investors evaluate startup opportunities.
    2 Startup Pitching: Learn how to communicate startup value effectively.
    3 Business Evaluation: Analyze market, execution, and revenue potential.
    4 Application Development: Build complete business-focused web applications.

# PROMPT:
You are an expert full-stack developer and product designer.

Build a complete, production-quality AI Shark Tank Simulator as a single self-contained HTML file.

Requirements:

1. USER IDEA INPUT
- Startup Name
- Problem Statement
- Solution
- Revenue Model
- Target Audience
- Funding Ask

2. AI JUDGES
Create 4 distinct AI judges:

🦈 Venture Capitalist
- Focus on market size and scalability

🦈 Founder
- Focus on execution

🦈 Customer
- Focus on usefulness

🦈 Angel Investor
- Focus on profitability

3. PITCH ROUND
- Display startup pitch
- Each judge asks 2 questions
- User can answer
- Judges react dynamically

4. SCORING SYSTEM
Score out of 100:

- Market Potential
- Innovation
- Business Model
- Execution
- Investment Worthiness

5. INVESTMENT DECISION
Generate:
- Invest
- Reject
- Acquire
- Come Back Later

Show:
- Suggested Valuation
- Funding Amount
- Reasoning

6. UI
- Modern dark theme
- Shark Tank style
- Animated cards
- Responsive design

7. BONUS
- Confetti on funding success
- Download Pitch Report PDF
- Leaderboard
- Share Result button

Deliver as a single HTML file with no backend required.
